Paul Attie's Publications

  • Model and Program Repair via Group Actions and Structure Unwinding
    Paul Attie, William Cocke.
    ACM Transactions on Computational Logic (TOCL).
    April 2025, Volume 26, Issue 2, pp. 1 - 44
    Abstract / Preprint (PDF) / Bibtex
    Synopsis: Presents a method of applying symmetry reduction to model repair and concurrent program repair. Shows that "unwinding" a structure can genreate repairs that do not exist in the original unwound structure. Gives closed-form results for the reductions in number of states in the Kripke structure that can be achieved by symmetry reduction.

  • Operational Annotations: A New Method for Sequential Program Verification
    Paul Attie.
    NASA Formal Methods: 14th International Symposium, NFM 2022, Pasadena, CA, USA, May 24-27, 2022, Proceedings Pages 597 - 615.
    Abstract / Preprint (PDF) / Bibtex
    Synopsis: I present a new method for specifying and verifying the partial correctness of sequential programs which uses abstract programs instead of formulae to define pre- and post-conditions.

  • From global choreographies to verifiable efficient distributed implementations
    Mohamad Jaber, Ylies Falcone, Paul Attie, Al-Abbass Khalil, Rayan Hallal, Antoine El-Hokayem.
    Journal of Logical and Algebraic Methods in Programming.
    October 2020, Volume 115, pp. 1 - 24
    Abstract / Preprint (PDF) / Bibtex
    Synopsis:Presents a method to automatically synthesize efficient distributed implementations from high-level global choreographies. Also defines a translation of the distributed implementations to equivalent Promela versions, which enables model-checking in SPIN. Implemented a Java prototype.

  • Global and Local Deadlock Freedom in BIP
    Paul C. Attie, Saddek Bensalem, Marius Bozga, Mohamad Jaber, Joseph Sifakis, Fadi A. Zaraket.
    ACM Transactions on Software Engineering Methodology (TOSEM).
    January 2018, Volume 26, Issue 3, pp. 1 - 48
    Abstract / Preprint (PDF) / Bibtex
    Synopsis: Presents a sound and complete criterion for deadlock freedom. The criterion can be verified in small subsystems, in which case it becomes incomplete. It is the first sound and complete characterization of deadlock in models of concurrency with nondeterministic local choice of actions, e.g., CCS, CSP, I/O Automata, BIP.

  • Model and Program Repair via SAT Solving
    Paul C. Attie, Kinan Dak Al Bab, and Mohamad Sakr
    ACM Transactions on Embedded Computing Systems (TECS),
    December 2017, Volume 17, Issue 2, pp. 1-25
    Abstract / Preprint (PDF) / Technical report (longer than preprint) / Bibtex
    Synopsis: Presents a method for the repair of Kripke structures and concurrent programs so that they satisfy a specification in the temporal logic CTL. The method is implemented in the GUI-based tool Eshmun.

  • Finite-state concurrent programs can be expressed succinctly in triple normal form
    Paul C. Attie,
    Information Processing Letters (IPL), Vol. 123, July 2017, Pages 8-13.
    Abstract / Preprint (PDF) / Bibtex
    Synopsis: Shows that any concurrent program can be succinctly rewritten in triple normal form (all variables shared by exactly three processes), up to strong bisimulation. Hence restriction to triple normal form does not incur any loss of expressiveness.

  • Synthesis of Large Dynamic Concurrent Programs from Dynamic Specifications
    Paul C. Attie.
    Formal Methods in System Design, April 2016, Volume 48, Issue 1, pp. 94–147.
    Abstract / Published version / Preprint / Bibtex
    Synopsis: Presents pairwise normal form (all variables shared by exactly two processes) and gives an efficient (polynomial in the number of processes) method for synthesizing large concurrent programs from CTL specifications, using pairwise normal form. Shows that concurrent programs in pairwise normal form inherit correctness properties from each pair-system, i.e., each pair of interacting processes executing in isolation. Does not assume that processes are similar.
