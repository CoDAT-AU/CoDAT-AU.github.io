Model Check

You can model check any model after inputing its specifications, to do so simply go to the Tools Menu -> Model Check (F8). Model Check will be carried out for the entire open structure.

After model checking is complete, the result is displayed in the status panel (bottom right corner of the screen), with the time required in nano seconds.
